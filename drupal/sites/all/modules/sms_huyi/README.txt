sms_huyi
~~~~~~~~~~

sms_huyi is a drupal module to add the http://ihuyi.com/ gateway to the sms framework module.

INSTALL
~~~~~~~

1. See the getting started guild on installing drupal modules:
http://drupal.org/getting-started/install-contrib/modules

2. Go to admin/smsframework/gateways to setup the twilio gateway.
